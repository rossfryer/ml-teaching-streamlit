# HSC Machine Learning Teaching Lab

A multi-page Streamlit app for teaching Year 12 HSC Software Engineering machine learning concepts.

## Included pages

- Home launcher
- Linear Regression
- Polynomial Regression
- Logistic Regression

## Run

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## Purpose

The app guides students through:

- dataset inspection
- feature and target selection
- train/test split
- model training
- visualisation
- prediction
- evaluation metrics
- HSC-style written explanations
